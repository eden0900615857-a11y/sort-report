## 氣泡排序法

氣泡排序（Bubble Sort）是一種簡單的交換排序。透過反覆比較相鄰元素，若順序錯誤則交換，讓比較大的元素像氣泡一樣慢慢浮到序列末端。

- 原理：比較相鄰元素，若前者大於後者則交換；每一輪把最大值放到最後。
- 最好情況：O(n)（已排序時，只需一次比較）
- 平均情況：O(n²)
- 最壞情況：O(n²)

## 選擇排序法

選擇排序（Selection Sort）是一種簡單的排序方法，每次從未排序區間選出最小（或最大）元素，並將其放到已排序區間的末端。

- 原理：將整個陣列分成已排序和未排序兩部分，每次在未排序部分選出最小元素交換到前面。
- 最好情況：O(n²)
- 平均情況：O(n²)
- 最壞情況：O(n²)

## 插入排序法

插入排序（Insertion Sort）將資料分成已排序和未排序兩部分，逐一將未排序元素插入到已排序部分的正確位置。

- 原理：從第二個元素開始，將當前元素與前面的已排序區間比較，插入適當位置。
- 最好情況：O(n)（已排序時）
- 平均情況：O(n²)
- 最壞情況：O(n²)

## 合併排序法

合併排序（Merge Sort）是一種分治演算法，先遞迴地將陣列分成兩半，排序後再合併成單一已排序序列。

- 原理：分而治之，先將陣列分成小片段，再逐步合併成已排序陣列。
- 最好情況：O(n log n)
- 平均情況：O(n log n)
- 最壞情況：O(n log n)

## 綜合比較

比較各排序演算法的特性與適用場合：

- 氣泡排序：實作簡單，適合教學與小型資料，不適合大量資料。
- 選擇排序：交換次數較少，但比較次數仍為 O(n²)，適合理解選擇過程。
- 插入排序：對近乎已排序資料表現較佳，適合小型陣列與線上資料插入場景。
- 合併排序：穩定且時間複雜度穩定為 O(n log n)，適合大型資料排序，但需額外空間。
