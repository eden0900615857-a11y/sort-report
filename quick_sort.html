💡 什麼是快速排序法？

快速排序法是一種高效的分治排序演算法。它選擇一個 pivot 作為基準，將陣列分成小於與大於 pivot 的兩個子區間， 然後對這兩個區間遞迴執行相同操作，最終合併成已排序結果。

⏱️ 時間複雜度

最好情況： O(n log n)
平均情況： O(n log n)
最壞情況： O(n²) - 當 pivot 選得非常不均衡時 

function quickSort(arr)
  if arr.length <= 1 return arr
  pivot = arr[arr.length - 1]
  left = []
  right = []
  for each value in arr[0..end-1]
    if value < pivot then left.push(value)
    else right.push(value)
  return quickSort(left) + [pivot] + quickSort(right)
